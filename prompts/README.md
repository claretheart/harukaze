# prompts/ — AIプロンプト集

効いたプロンプトを「資産」として貯める場所。次も同じ品質を出すため。

## 入れるもの（例）
- `image/` … 画像生成プロンプト（実写風カフェ、水彩背景 など）
- `writing/` … 文章生成（事例紹介文、SNS、note）
- `research/` … リサーチ用プロンプト

## 書き方
1ファイル1用途。冒頭に「狙い」、本文にプロンプト、末尾に「結果メモ／改善点」。

## 例（画像・実写風カフェ）
> 京都の伝統的な町家を改装した小さなカフェの店内。自然光、木の質感、コーヒーカップ、和モダン、実写、人物なし、文字なし。
