# SNS投稿 テンプレート

> **使い方**：〔業種〕を埋めて投稿文を作る。プラットフォーム：Instagram / X / Threads 想定。
> harukaze の声で。**売り込まない・煽らない・AIっぽくしない**。

---

## 変数

| 変数 | 例 |
|------|----|
| 業種 | 〔業種〕 |
| 店名 | 〔店名〕 |
| 今日の主役 | 〔商品/シーン〕 |
| 伝えたい想い | 〔ひとこと〕 |

---

## 投稿タイプ別ひな型

### ① 体験・風景型（世界観を届ける）
```
〔店名〕の〔商品/シーン〕。
〔ひとこと〕。

京都の小さな〔業種〕から、今日も少しだけ春風を。
```

### ② Tips型（役に立つ）
```
〔業種〕さん向け・小さなデジタルのコツ。

「〔よくある悩み〕」
→ 〔やさしい解決の一歩〕

むずかしく考えなくて大丈夫。少しずつで十分です。
```

### ③ 問いかけ型（距離を縮める）
```
〔業種〕をされている方へ。
〔悩み〕で止まっていませんか？

「こんなこと聞いてもいいのかな」も大歓迎です。
よかったら、ひとことだけでも。
```

### ④ お知らせ型（事例公開など）
```
あたらしい提案事例を公開しました 🌸
京都の〔業種〕「〔店名〕」（イメージ）

〔ひとこと〕
くわしくはプロフィールのリンクから。
```

---

## ルール

- 1投稿1テーマ。最初の一文でそっと引く（煽らない）
- 絵文字は控えめ（🌸 程度）。多用しない
- ハッシュタグは **1〜2個**（例：#京都 #〔業種〕）。盛らない
- 文末は静かに。「！」連打や「今だけ」「絶対」は禁止
- 投稿前チェック：**「読んだ人がやさしい気持ちになるか？」**
