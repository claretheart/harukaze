# brand/ — ブランドの核

harukaze の「憲法」。判断に迷ったとき、ここを基準にします。

## 中身
- `BRAND.md` … ミッション・ブランドイメージ・禁止事項・ゴール
- `DESIGN_SYSTEM.md` … カラー・フォント・デザイン方針
- `docs/` … personas（ターゲット）／services（提供）／vision（想い）
- `logo/` … harukaze ロゴ（円形）

## カラー
背景 `#FAF7F2` ／ 桜色 `#E7C9C9` ／ 若草 `#B8C9A3` ／ ダークブラウン `#4A403A`

## フォント
本文 Noto Sans JP ／ 見出しに Shippori Mincho・Zen Maru Gothic（事例ページ）

## 原則
迷ったら「制作会社」ではなく「**京都の案内人**」を優先する。
