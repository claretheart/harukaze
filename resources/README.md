# resources/ — 共有素材

複数プロジェクトで使い回す素材の置き場。

## 中身
- `photos/` … 写真素材（実写・AI生成）。`photos/cafe/` に珈琲 春灯の原本など
- `illustrations/` … イラスト・水彩素材（名刺背景 `card-bg.png` など）
- `icons/` … アイコン・QR（`qr-site.png` など）

## ルール
- 商用利用可否・出典が分かるようにする
- 特定プロジェクト専用の素材は、そのプロジェクト内に置く（ここは「共有」用）
- Web用は軽量化してから各プロジェクトへ（原本はここ、軽量版はプロジェクト側）
